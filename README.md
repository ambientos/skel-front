# Skeleton front

Include:

- Babel
- Webpack
- browser-sync
- gulp
- gulp-autoprefixer
- gulp-notify
- gulp-plumber
- gulp-pug
- gulp-sass
- gulp-sourcemaps

To start the project you need to install npm packages:

```sh
$ yarn
```

To run the project:

```sh
$ gulp
```

To build js into production:

```sh
$ yarn webpack-build
```
